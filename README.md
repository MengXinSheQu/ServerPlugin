# ServerPlugin - 服务器轻型插件💻[中文]

## **介绍:**

# 该插件向服务器添加了以下功能:
- 感应卡交互(门 物品柜 电板)
- 感应卡黑名单列表(API)
- SCP-207不掉血
- SCP-1853与SCP-207不中毒
- 无限子弹
- 自定义SCP-330拾取上限
- 去除迷雾
- 更多的聊天指令

以上功能可进行设置

## **使用:**

将`ServerPlugin.dll`放至`EXILED\Plugins`内

服务器就会加载该插件了

## **配置:**


```
IsEnabled: true #启动该插件

Debug: false #启动DeBug(暂无效)

DisablePoison: true #关闭SCP-1853与SCP-207的中毒 🤢❌

DisableSCP207Hurt: true #关闭SCP-207掉血 🥤👍

InfAmmo: true #无限子弹 🔫

NoFog: true #无迷雾 🌫

Candys: 2 #最大拿糖数量 🍬

RemoteKeycard: true #启动免手持卡 🎫

RemoteKeycard_Scp: false #免手持卡应用于SCP 💀🎫

SpecatorChat: false #旁观者可使用BC指令 ☠📣

MuteChat: false #被禁言玩家可使用聊天指令 🤐📣
```
